## 📝 Creating your first keybind

You create a script to tell your computer what to do. A script is a text file with instructions. 

1. Right-click on your desktop.
2. Select New from the menu.
3. Choose the option to create a text file.
4. Name the file "myscript.ahk". Make sure the extension ends in .ahk.
5. Right-click the new file.
6. Select edit to open the file in Notepad.
7. Type a simple instruction, such as: `a::b`.
8. Save and close the file.
9. Double-click your file icon to run it.

When you press the A key on your keyboard, the computer acts as if you pressed the B key. This demonstrates the core function of the program.
